I'M PUTRI!

ditulis oleh Putri Soleha pada 12 oktober 2020

Nama saya Putri Soleha Nur Assyifa, saya lulusan SMK Negeri 4 Bogor jurusan rekayasa perangkat lunak. Saya memiliki pengalaman membuat aplikasi berbasis web,di sekolah bahasa pemrograman yang saya pelajari adalah HTML, CSS, JavaScript, dan PHP dengan menggunakan framework CI.

Saya memiliki pengalaman magang di instansi pemerintahan yaitu Kantor Imigrasi Bogor, saya ditempatkan di divisi teknologi, komunikasi, dan informasi. Tugas yang saya lakukan saat itu adalah melayani masyarakat yang ingin membuat pasport juga menjelaskan mengenai penggunaan aplikasi layanan pasport online.

Website ini adalah website sederhana yang saya buat menggunakan HTML dan CSS, memang masih banyak kekurangannya, namun saya ingin terus mengembangkan skill dan belajar.
